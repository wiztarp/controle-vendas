# Controle de Vendas

Sistema de controle de vendas desenvolvido em **C#** para gerenciamento de produtos, estoque e registro de vendas.

## Tecnologias utilizadas

- C#
- .NET
- Windows Forms
- SQL Server

## Funcionalidades

- Cadastro de produtos
- Controle de estoque
- Registro de vendas
- Consulta de vendas

## Objetivo do projeto

Este projeto foi desenvolvido com o objetivo de praticar conceitos de desenvolvimento de aplicações desktop utilizando C# e banco de dados, simulando um sistema simples de controle de vendas utilizado em pequenas empresas.
